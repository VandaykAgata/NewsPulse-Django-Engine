# NewsPulse Engine 🚀

Полнофункциональный новостной портал на Django с поддержкой асинхронных задач, подписок и мультиязычности.

## ✨ Ключевые особенности:
* **Асинхронность (Celery + Redis):** Уведомления о новых постах и еженедельные рассылки выполняются в фоновом режиме, не замедляя работу сайта.
* **Безопасность:** Конфиденциальные данные (ключи API, пароли БД и почты) полностью вынесены в переменные окружения (`.env`).
* **Система подписок:** Пользователи могут подписываться на категории и получать уведомления только о том, что им интересно.
* **Логирование:** Настроено продвинутое логирование в файлы (`general.log`, `errors.log`, `security.log`) для мониторинга состояния системы.
* **i18n:** Полная поддержка мультиязычности (русский/английский).

## 🛠 Технологический стек:
* Python 3.13 / Django 5.x
* PostgreSQL (основная БД)
* Redis (брокер сообщений)
* Celery (очередь задач)
* Django-allauth (авторизация через соцсети)

🇷🇺 Описание проекта (RU)

Этот проект — современный новостной портал на Django, усиленный искусственным интеллектом. Мы интегрировали нейросеть для автоматизации работы редактора: теперь система сама анализирует каждую новость.

Что было сделано:
Интеграция с OpenRouter API: Подключена модель Gemini 2.0 Flash для мгновенного анализа текста.

Автоматическая обработка: При сохранении новости ИИ определяет:

Тональность (позитивная, негативная, нейтральная).

Категорию (автоматическое присвоение тематики).

Summary (краткое содержание статьи для экономии времени читателя).

Безопасность: Реализована работа с переменными окружения через .env для защиты API-ключей.

Проблемы и решения:
Проблема 404/Connection: Бесплатные модели часто меняют адреса или перегружены.

Решение: Настроен динамический перебор моделей и добавлены обязательные заголовки (HTTP-Referer).

Проблема Таймаутов: Большие статьи долго обрабатывались и обрывали соединение.

Решение: Оптимизирована отправка данных (срез первых 700 символов) и увеличен таймаут до 30 секунд.

Проблема Парсинга: ИИ присылал ответ в Markdown, что вызывало ошибки JSON.

Решение: Написан парсер-очиститель, удаляющий лишние символы перед сохранением в базу.

🇺🇸 Project Overview (EN)

A Django-based news portal featuring deep AI integration for automated content analysis and categorization.

Key Features:
AI Engine: Integrated with OpenRouter API (Gemini 2.0) to analyze articles on the fly.

Metadata Generation: Automatically generates sentiment, category, and concise summaries.

Robust Backend: Custom error handling for API timeouts and invalid JSON responses.

Technical Challenges:
Path Management: Resolved issues with .env file detection using absolute paths in Python.

Prompt Engineering: Optimized AI prompts to ensure responses are returned in a strict JSON format for seamless database integration.

## 🚀 Как запустить:
1. Клонируйте репозиторий.
2. Создайте файл `.env` на основе примера и заполните свои данные.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Запустите миграции: `python manage.py migrate`.
5. Запустите Celery-воркер: `celery -A NewsPaper worker -l info`.
