# NewsPulse Engine 🚀

Полнофункциональный новостной портал на Django с поддержкой асинхронных задач, подписок и мультиязычности.

## ✨ Ключевые особенности:
* **Асинхронность (Celery + Redis):** Уведомления о новых постах и еженедельные рассылки выполняются в фоновом режиме, не замедляя работу сайта.
* **Безопасность:** Конфиденциальные данные (ключи API, пароли БД и почты) полностью вынесены в переменные окружения (`.env`).
* **Система подписок:** Пользователи могут подписываться на категории и получать уведомления только о том, что им интересно.
* **Логирование:** Настроено продвинутое логирование в файлы (`general.log`, `errors.log`, `security.log`) для мониторинга состояния системы.
* **i18n:** Полная поддержка мультиязычности (русский/английский).

## 🛠 Технологический стек:
* Python 3.13 / Django 5.x
* PostgreSQL (основная БД)
* Redis (брокер сообщений)
* Celery (очередь задач)
* Django-allauth (авторизация через соцсети)

## 🚀 Как запустить:
1. Клонируйте репозиторий.
2. Создайте файл `.env` на основе примера и заполните свои данные.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Запустите миграции: `python manage.py migrate`.
5. Запустите Celery-воркер: `celery -A NewsPaper worker -l info`.
